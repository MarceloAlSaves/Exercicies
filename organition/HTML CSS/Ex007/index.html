Principais tipos:

Negrito/Italico

Para criar um testo em negrito se usa a tag b(nao semantica)

Para criar um testo em italico se usa a tag i(nao semantica)

Para criar um texto De destaque se usa a tag mark

Para criar um texto sublinhado se usa a tag u(nao semantica)

Para criar um texto riscado de não condizente se usa a tag del

Para criar um numero de cima se usa a tag sup: exp: X10+14

Para criar um numero de baixo se usa a tag sub: exp: H2O

Dica: Use control + Shift + P para abrir o menu de comandos rapido (parecido com o obsidian)
